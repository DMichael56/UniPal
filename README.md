# UniPal

A Flutter based university app developed by **Team 1D SUB** as part of our university project for Software engineering Theory and Practice.
UniPal allows students to book rooms, browse university buildings, chat in groups, and manage their university studying from using the app
--

## Features

- **Room Booking** — A step by step booking flow with building and room selection, date/time picker, and confirmation
- **Building Selection** — Browse all university buildings with amenity info (café, elevator, helpdesk) and room counts loaded from live data
- **Room Layout** — Visual seat grid for selected rooms
- **Group Chat** — Join and message study groups
- **Profile** — View your student profile
- **My Bookings** — View and manage your existing bookings

---

## Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (Dart SDK ^3.9.0)
- Android Studio or VS Code with Flutter extension
- A connected device or emulator

### Installation

```bash
git clone https://github.com/DMichael56/UniPal.git
cd UniPal
flutter pub get
flutter run
```
